Kırtasiye E-Ticaret Web Sitesi

Bu proje, PHP ve MySQL kullanılarak geliştirilmiş basit bir kırtasiye web sitesidir. Temel olarak ürün yönetimi işlemlerini içermektedir.

Özellikler
Ürün ekleme (Create)
Ürün listeleme (Read)
Ürün silme (Delete)
MySQL veritabanı ile veri yönetimi
Kullanılan Teknolojiler
PHP
MySQL
HTML
CSS
Amaç

Bu proje, temel CRUD işlemlerini öğrenmek ve web geliştirme pratiği kazanmak amacıyla geliştirilmiştir.

Kurulum
Projeyi indir veya klonla
Veritabanını oluştur (MySQL)
Gerekli tabloyu içe aktar
Projeyi localhost üzerinde çalıştır
